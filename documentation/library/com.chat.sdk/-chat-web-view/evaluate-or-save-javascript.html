evaluateOrSaveJavascript

private fun evaluateOrSaveJavascript(javascript: String, resultCallback: ValueCallback<String?>?): Unit

This method evaluates a javascript code and invokes the given resultcallback (if present). If the view is not ready the javascript and the callback will be added to the pending actions, and will be invoked after the view is ready.

Parameters

javascript - Javascript to evaluate.

resultCallback - Callback after javascript has been evaluated. Optional.
